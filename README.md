# Installation

Install [node.js](https://nodejs.org/en/download/) first

`npm install`

Assure [MongoDB](https://hub.docker.com/_/mongo/) is running on localhost (default setting)
and user and db credentials are correctly written in config.js

Then, run
`npm start`
For Express server


Or run
`npm run parse`
For DB fulfilling
